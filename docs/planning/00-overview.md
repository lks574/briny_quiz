# 개요

Open Trivia DB를 이용한 SwiftUI(iOS 17+) 퀴즈 앱을 만든다.

## 목표
- 문제 풀이 경험이 빠르고 직관적이어야 한다.
- 네트워크/데이터/라우팅 구조를 표준화한다.
- 캐시/에러/취소 흐름이 일관되게 동작한다.
- 스플래시/대시보드/퀴즈/결과/히스토리 흐름을 갖는다.
 - 탭 구조(대시보드/히스토리)로 UX를 단순화한다.

## 현재 상태
- 프로젝트 뼈대(Tuist, MVI, Router, Networking/Repository 스킬) 구성 완료
- 요구사항 상세는 확정되지 않음
