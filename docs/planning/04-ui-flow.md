# UI/플로우

## 기본 플로우
스플래시 -> (탭) 대시보드/히스토리 -> 퀴즈 진행 -> 결과

## 화면 초안
- 스플래시
  - 로고 표시
  - 초기 설정값 로딩
- 대시보드(퀴즈 선택 화면, 탭)
  - 문제 수: 10 고정
  - 난이도 선택 (easy/medium/hard)
  - 질문 유형 선택 (혼합/객관식/주관식)
  - 카테고리: 전체
  - (선택) 중복 방지 토큰 사용 여부는 내부 정책으로 고정
  - 시작 버튼
- 퀴즈 진행
  - 문제 텍스트
  - 보기 선택
  - 타이머(문제당 10초)
  - 스킵 버튼
  - 즉시 피드백(정답/오답 표시)
  - 진행 상태(현재/전체)
  - 다음 버튼
- 결과(대시보드 스택)
  - 점수
  - 다시 시작
- 히스토리(탭)
  - 내부 데이터 기반 기록 조회
