# Settings 탭 (팩 업데이트)

## 목적
- 오프라인 팩 업데이트를 사용자가 수동으로 수행
- 다운로드 진행/결과를 명확히 표시

---

## 1) 화면 구성
- 섹션: "오프라인 팩"
  - 현재 팩 버전 표시
  - "업데이트 확인" 버튼
  - 업데이트 가능 시 "다운로드" 버튼 활성화
  - 다운로드 진행률 표시

---

## 2) 동작 흐름
1. "업데이트 확인" 버튼
   - 서버 메타(`packVersion.json` 등) 조회
   - 최신 버전이면 "최신" 상태 표시
   - 최신 아니면 "다운로드" 버튼 활성화

2. "다운로드" 버튼
   - 진행률 표시
   - 완료 후 로컬 저장
   - 현재 팩 버전 갱신

---

## 3) 오류 처리
- 네트워크 실패: "네트워크 오류" 메시지
- 용량 부족: "저장 공간 부족" 메시지
- 파일 검증 실패(해시): "다운로드 파일 손상" 메시지

---

## 4) 로컬 저장 규칙
- 다운로드 파일은 `Application Support/QuizPack/pack_v{version}.json`에 저장
- `current.json`에 현재 버전 기록
- 저장 완료 후 자동 최신 팩으로 선택

---

## 5) 비고
- 서버 스펙 확정 전까지 메타 조회/다운로드는 stub 처리 가능
