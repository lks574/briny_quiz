# 요구사항

## 확정 내용
- 문제 수: 10
- 난이도: 사용자가 선택 (easy/medium/hard)
- 카테고리: 전체
- 질문 유형: 사용자가 선택 (혼합/객관식/주관식)
- 게임 규칙
  - 타이머: 문제당 제한시간
  - 제한시간: 10초
  - 점수: 정답 1점
  - 오답 처리: 즉시 피드백
  - 스킵: 허용
- 결과 화면
  - 점수
  - 재시작
- 히스토리 화면
  - 내부 데이터 기반 기록 조회
- 오프라인/캐시: 필요
- 라우팅/탭 구조: 탭 2개(대시보드/히스토리), 퀴즈/결과는 대시보드 스택

## 기본 가정(변경 가능)
- iOS 17+ 대상
- SwiftUI + MVI(Store) 구조
- 네트워크는 URLSession + async/await
